Frequently Asked Questions

 

Q: I have used Chromaseq and Phred and Phrap to produce a sequence. I've also made more base calls using Chromaseq's chromatogram viewer. However, one part of that sequence isn't the best, and I have resequenced part of the gene. Can I add these new chromatogram to the existing contig?

A: Not yet. You will need to reprocess the original chromatograms again for that sequence, and remake any base calls. We will add the ability to add new reads in the future.

 

Q: Can I insert bases into a sequence?

A: Yes, using the Insert gaps tool.

 

Q: I would like to use Chromaseq to make my base calls for a matrix of sequences, then align the matrix. Can I do this?

A: Yes, but only if you do the alignment from within Mesquite so that it is fully aware of the re-alignment at it happens. For example, you can use the tools in the Matrix>Align Multiple Sequences submenu.

 

Q: Can I reverse complement the sequences in the editor and still have Chromaseq function to let me look at the contigs and chromatograms?

A: Yes, if you use the Matrix>Alter/Transform>Reverse Complement to reverse and complement the entire sequence. If you reverse complement parts of the sequence, or use a tool outside of Mesquite, the connections between the bases in the editor and the bases in the chromatograms will be lost.
