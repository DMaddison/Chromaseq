Primer Sequences File

Chromaseq can search for primer sequences in your chromatograms, and mark the location of these sequences. This will allow you to remove the primers from your final sequences, should you so choose. However, you will need to create a file containing the primer sequences. The format of this file is very simple:

<primer name><tab><primer sequence>
<primer name><tab><primer sequence>

Here is an example file:

18Sai CCTGAGAAACGGCTACCACATC
18Sbi GAGTCTCGTTCGTTATCGGA
