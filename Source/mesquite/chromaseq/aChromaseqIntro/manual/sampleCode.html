If you wish to have the sample codes that are contained in the chromatogram file names translated into other names (for example, the full names of the species or samples), you will need to provide a sample codes & names file.

The format of the sample codes & names file is:

<sample code><tab><short sample name><tab><long sample name>;
<sample code><tab><short sample name><tab><long sample name>;
<sample code><tab><short sample name><tab><long sample name>;
<sample code><tab><short sample name><tab><long sample name>;

The sample code is the code present in the chromatogram file name. The short sample name is the name to which this will be translated in some places in the Phred/Phrap/Mesquite system. The long sample name is the name to which it will be translated to in the sequence names of process sequences. The long name is optional. The short name is used for file names, to ensure that file names are not longer the maximum file name length of the operation system.

Here is part of an example file:

1120<tab>Bemb perspicuum AZ 1120<tab>Bembidion perspicuum AZ 1120 ;
1173<tab>Bemb antiquum ON 1173<tab>Bembidion antiquum ON 1173 ;
1250<tab>Bemb louisella NS 1250<tab>Bembidion louisella NS 1250 ;
1514<tab>Bemb antiquum NC 1514<tab>Bembidion antiquum NC 1514 ;
1598<tab>Bemb rufotinctum NC 1598<tab>Bembidion rufotinctum NC1598 ;
1626<tab>Bemb antiquum IA 1626<tab>Bembidion antiquum IA 1626 ;
1856<tab>Bemb paraenulum VA 1856<tab>Bembidion paraenulum VA 1856 ;
1960<tab>Bemb rothfelsi MO 1960<tab>Bembidion rothfelsi MO 1960 ;
2320<tab>Bemb perspicuum CA 2320<tab>Bembidion perspicuum CA 2320 ;
