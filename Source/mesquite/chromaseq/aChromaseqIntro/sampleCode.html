Sample Codes & Names File

If you wish to have the sample codes that are contained in the chromatogram file names translated into other names (for example, the full names of the species or samples), you will need to provide a sample codes & names file.

There are two allowed formats for the sample codes & names file, tab-delimited and space-delimited. In both formats, each line of the file contains the information for one sample code. Each line consists of four parts: (1) the sample code, (2) a short name of the sample, (3) a long name of the sample, and (4) an ending semicolon. The sample code is the code present in the chromatogram file name. The short sample name is the name to which this will be translated in some places in the Phred/Phrap/Mesquite system. The long sample name is the name to which it will be translated to in the sequence names of process sequences. The long name is optional. The short name is used for file names, to ensure that file names are not longer the maximum file name length of the operation system. The two formats differ in the character that separates the parts (a tab or space) and the structure of the first three elements.

Delimited by tabs

In this format, the elements on each line are separated by tabs (), including a tab between the long sample name and the semicolon:

<sample code><short sample name><long sample name>;
<sample code><short sample name><long sample name>;
<sample code><short sample name><long sample name>;

Here is part of an example file:

1839Bemb honestum VA 1839Bembidion honestum VA:Wingina 1839;
1840Bemb honestum VA 1840Bembidion honestum VA:New Canton 1840;
1841Bemb honestum VA 1841Bembidion honestum VA:New Canton 1841;
1842Bemb arenobilis NC 1842Bembidion arenobilis NC:Bynum 1842;
1843Bemb arenobilis VA 1843Bembidion arenobilis VA:Brookneal 1843;
2082Amerizus spectabile CA 2082Amerizus spectabile CA 2082;
2083Bemb complanulum BC 2083Bembidion complanulum BC 2083;
2084Bemb farrarae BC 2084Bembidion farrarae BC 2084;
2085Bemb improvidens BC 2085Bembidion improvidens BC 2085;
2086Bemb mimus IA 2086Bembidion mimus IA:Manchester 2086;
2087Bemb platynoides OR 2087Bembidion platynoides OR 2087;

In this format, the elements can consist of multiple tokens.

Delimited by spaces

In this format, the elements on each line are separated by spaces; there does not need to be a space between the long sample name and the semicolon:

<sample code> <short sample name> <long sample name>;
<sample code> <short sample name> <long sample name>;
<sample code> <short sample name> <long sample name>;

Each element in this format must consist of exactly one token. If the sample name would naturally consist of more than one token, it must be "tokenized" by surrounding it by single quotes, and converting any contained single quotes into two consecutive quotes. Here is part of an example file:

1839 'Bemb honestum VA 1839' 'Bembidion honestum VA:Wingina 1839' ;
1840 'Bemb honestum VA 1840' 'Bembidion honestum VA:New Canton 1840' ;
1841 'Bemb honestum VA 1841' 'Bembidion honestum VA:New Canton 1841' ;
1842 'Bemb arenobilis NC 1842' 'Bembidion arenobilis NC:Bynum 1842' ;
1843 'Bemb arenobilis VA 1843' 'Bembidion arenobilis VA:Brookneal 1843' ;
2082 'Amerizus spectabile CA 2082' 'Amerizus spectabile CA 2082' ;
2083 'Bemb complanulum BC 2083' 'Bembidion complanulum BC 2083' ;
2084 'Bemb farrarae BC 2084' 'Bembidion farrarae BC 2084' ;
2085 'Bemb improvidens BC 2085' 'Bembidion improvidens BC 2085' ;
2086 'Bemb mimus IA 2086' 'Bembidion mimus IA:Manchester 2086' ;
2087 'Bemb platynoides OR 2087' 'Bembidion platynoides OR 2087' ;
